# react-native-tiktok-business-kit

A React Native wrapper for TikTok Business SDK. Easily track events, manage user identifiers, handle attribution, and integrate TikTok marketing analytics in your React Native applications with a unified TypeScript API.

## Installation


```sh
npm install react-native-tiktok-business-kit react-native-nitro-modules

> `react-native-nitro-modules` is required as this library relies on [Nitro Modules](https://nitro.margelo.com/).
```


## Usage


```js
import { multiply } from 'react-native-tiktok-business-kit';

// ...

const result = multiply(3, 7);
```
